# Release v0.0.4

## 更新内容

- 新增多窗口支持，可同时打开多个 CoPaw 窗口
- 多实例自动共享后端服务，避免端口冲突
- 优化启动速度和内存占用
- 修复后端服务生命周期管理问题

## 安装包

| 架构 | 文件名 | SHA256 |
|------|--------|--------|
| Apple Silicon | CoPaw-arm64.dmg | `待填写` |
| Intel x86_64 | CoPaw-x86_64.dmg | `待填写` |

## 系统要求

- macOS 10.15 (Catalina) 或更高版本

## 安装方法

1. 下载对应架构的 DMG 文件
2. 双击挂载 DMG
3. 将 CoPaw 拖拽到 Applications 文件夹
4. 首次运行需在 系统设置 → 隐私与安全性 中允许打开

## 验证下载

```bash
# 计算 SHA256
shasum -a 256 CoPaw-arm64.dmg
shasum -a 256 CoPaw-x86_64.dmg
```
